​		user_professional_title



请求方式：POST

```json
{
	"apiVersion": "string",
	"channel": "string",
	"postData": {
        "titleId":"职称Id",
        "userId":"用户Id",
        "examScore":"认证考试成绩"
	}
}
```

```json
    "data":true
```
